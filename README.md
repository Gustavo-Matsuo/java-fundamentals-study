# ☕ java-fundamentals-study

## 📌 Sobre o projeto:
Repositório com exercícios e práticas desenvolvidas durante o aprendizado de Java, abordando conceitos fundamentais da linguagem.

## 📚 Conteúdo abordado:
- Manipulação de variáveis
- Estruturas condicionais
- Testes de conversão de dados
- Manipulação de caracteres
- Simulação de sistema bancário simples

## 🧠 Conceitos aplicados:
- Programação Orientada a Objetos (POO)
- Herança (classe mãe e classe filha)
- Estruturas de decisão

## ▶️ Como executar:
1. Clonar o repositório  
2. Abrir em uma IDE  
3. Executar as classes individualmente  

## 📈 Objetivo:
Consolidar os fundamentos da linguagem Java através de exercícios práticos.
